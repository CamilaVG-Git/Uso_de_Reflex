Planteamiento del problema
Se requiere que el estudiante desarrolle su primera página web utilizando el framework Reflex, guiándose exclusivamente por la documentación oficial. Posteriormente, deberá documentar todo el proceso y las conclusiones obtenidas.

Adicionalmente, el proyecto deberá ser publicado en GitHub, incluyendo un archivo README que explique cómo ejecutar la aplicación en cualquier máquina.

Objetivos
Familiarizarse con el uso de Reflex para desarrollo web.
Comprender la estructura básica de una aplicación web.
Desarrollar habilidades de autoaprendizaje mediante documentación técnica.
Fortalecer la capacidad de análisis y redacción técnica.
Aprender a usar GitHub como plataforma de control de versiones y publicación de proyectos.
Instrucciones generales
El estudiante deberá:

Investigar y utilizar la documentación oficial de Reflex.
Desarrollar una aplicación web básica.
Documentar todo el proceso realizado.
Subir el proyecto completo a GitHub.
Incluir un archivo README.md con instrucciones claras para ejecutar el proyecto en cualquier computadora.
Requisitos del proyecto
La página web debe incluir como mínimo:

Un título principal.
Un texto de bienvenida.
Un botón interactivo (ejemplo: cambiar texto o estado al hacer clic).
Documentación (PDF)
El documento debe incluir:

Introducción del proyecto.
Pasos de instalación y desarrollo.
Problemas encontrados y soluciones.
Resultados obtenidos.
Conclusiones personales sobre el uso de Reflex.
Entregables
Código fuente del proyecto en GitHub.
Repositorio público.
Archivo README.md con:
Descripción del proyecto.
Instrucciones de instalación.
Cómo ejecutar la aplicación paso a paso.
Documento en formato PDF con la documentación solicitada.
Criterios de evaluación
Funcionamiento de la página web (30%)
Uso correcto de Reflex (20%)
Creatividad y personalización (20%)
Calidad de la documentación (20%)
Correcta implementación de GitHub y README (10%)


Más info:

--Requisitos--
- Python 3.10+

--Instalación--
pip install reflex

--Ejecutar el proyecto--
reflex init
reflex run

--Resultado--
La app se ejecuta en http://localhost:3000
